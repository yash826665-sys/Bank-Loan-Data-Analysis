# Bank Loan Analysis Dashboard

## About the Project

This project focuses on analyzing bank loan data to understand lending performance, loan quality, and customer borrowing trends. The analysis was done using **Python**, **SQL**, and **Power BI** to create an interactive dashboard that helps visualize important business metrics.

---

## Tools Used

* Python (Pandas, NumPy, Seaborn, Matplotlib)
* SQL Server
* Power BI
* DAX
* Excel

---

## What I Did

* Cleaned and prepared the dataset using Python.
* Used SQL to calculate important KPIs.
* Built an interactive Power BI dashboard.
* Analyzed loan performance, customer trends, and loan portfolio quality.

---

## Dashboard Includes

* Loan Applications
* Funded Amount
* Amount Received
* Good Loans vs Bad Loans
* Monthly Loan Trends
* State-wise Analysis
* Loan Purpose Analysis
* Loan Term Analysis
* Home Ownership Analysis
* Employee Length Analysis

---

## Key Insights

* Analyzed **38,576** loan applications.
* **86.2%** of loans were classified as Good Loans.
* Debt Consolidation was the most common loan purpose.
* California had the highest number of loan applications.
* Most borrowers preferred **36-month** loan terms.
